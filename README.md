## 프로젝트 설명
- 프로젝트를 진행하면서 아쉬웠던 부분들을 전반적으로 수정
- 프로젝트 완성후 알게 된 상속 개념 연습
- 깃허브 활용   

#
## 프로그램 설명
- 기존 작성했던 프로그램 기능 소개 짤막하게 작성

- 콘솔창에서 직원별 아이디와 비밀번호를 입력해서 로그인후 기능 사용
- 예시 사진
